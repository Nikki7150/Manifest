# Manifest - A Vision Board App

## Description
I always like to make vision boards at the start of every year to keep as my mac or ipad background to remind me of my goals and other stuff that I want to try to achieve in that year. I also like making collages with weirdly shaped and sized picutres and stickers without a perfect template. So i combined both of these and made this app to be able to make both. 
This is Manifest. It is an app where you can make vision boards, collages, etc with the in-app stickers, text option, drawing mode, and even uploading your own picutres. I have included many editing tools that I personally coded into the app, using the help of google and ChatGPT to bring the editing features to life. I did not copy paste the code into the app, obviously. I handwrote and changed th code to make it my own but I did use AI to learn how to code each filter and all the changes I could add using Fabric.js. You can also download the board right in the app in the highest resolution with every single picture, text, drawing, image, and sticker you have. You also have the option to change the background of the board itself to make it a white board, grip paper or dotted paper from the pre-existing cork board background. You can also group and ungroup elements, and send backward and bring forward all the elements according to your active selection. 
I also have added keyboard shortcuts so it makes it easy for everyone who uses other aps like canva and figma. You can use ctrl + c (or cmd + c) and ctrl + v (or cmd + v) for copy and paste. You can use ctrl + z ( or cmd + z) to undo and ctrl + y (or cmd + y) to redo. You can use the up arrow on the selected element to bring it forward and the down arrow on the selected element to send it behind. 
Your board gets saved in your local storage so you will never lose it if you accidentally reload the page. You will also not lose it if you close the website either. It will persist in your local storage so you can come back to it anytime. 
I also curated all the stickers, decor and frames from all over the internet and put the ones i thought were the best for everyone to use. I handpicked them, cropped every one of them, removed their background and renamed every single one so it is easy for enyone who wants to fork my code and change it up. I've put my all in this app because I have wanted to make something like this for a long time. 
This app took a lot of time and patience to make so I hope everyone likes it! Thank you and have fun!

Some of the features are listed below: 
- Text Editing
  	- Fonts
  	- Color
  	- send back
  	- bring forward
  	- group
  	- ungroup
- Image editing
  	- brightness slider
  	- contrast slider
  	- saturation slider
  	- blur slider
  	- grayscale button
  	- sepia button
  	- invert button
  	- reset filters
  	- send back
  	- bring forward
  	- group
  	- ungroup

P.S., when you group, you cannot make any changes to the filters and you wont see the editing box pop up because it is a group and no longer a single element anymore. To edit each element separately, ungroup the elements and change the filters separately and group them again. sorry!!
